My projects

One of my current projects is a JavaScript Pong Clock which displays the current time as the score of a pong game. It can be seen at pongclock.me

All of the graphics are drawn using JavaScript and HTML5 Canvas. Here's an example:


			this.draw = function() {
			context.fillStyle = "white";
			context.beginPath();
			context.rect(this.x,this.y,this.width,this.height);
			context.closePath();
			context.fill();
			}
